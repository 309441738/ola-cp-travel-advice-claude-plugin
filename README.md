# Cruise Travel Advice · Claude Code 插件

提供关于邮轮旅行的实用建议，包括行前准备、最佳出发时间及目的地推荐，帮助您规划完美的邮轮之旅。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/ola-cp-travel-advice-claude-plugin
/plugin install ola-cp-travel-advice@ola-cp-travel-advice-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Cruise Travel Advice**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
